# Split Circular Linked-List

Split Circular Linked-List having k elements.

## Exmaple

CLL : 1 -> 2 -> 3 -> 4 -> 5 -> 6 \
k = 2 \
After Split : \
1 -> 2 \
3 -> 4 \
5 -> 6 

CLL : 1 -> 2 -> 3 -> 4 -> 5 -> 6 \
k = 3 \
After Split : \
1 -> 2 -> 3 \
4 -> 5 -> 6 

## Features

- Create Circular Linked-List.
  - Insert at head
  - Insert at tail
- Print Circular Linked-List.
- Split Circular Linked-List having k elements.
